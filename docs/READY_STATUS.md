# Factomedia Studio readiness

## Ready for product review

The collaborator experience is implemented in `main` and the official GitHub Pages workspace is published at `/factomedia-group/mi-dia/`.

Included:

- research-backed Studio shell and minimal navigation;
- universal capture with preserved initial material;
- one visible next action per story;
- guided editorial creation and human-reviewed AI suggestions;
- command palette with keyboard shortcut;
- live newsroom and distribution entry points;
- social post tracking, imported-post origin, snapshots and alerts;
- responsive collaborator workspace;
- UX rationale and social architecture documentation.

The official static collaborator route was published directly to `gh-pages` so it does not depend on the stale exported page that was previously being served. The Next.js source remains in `main` as the canonical implementation.
